Bot visibility — required Discord permissions ▼
Problem: the bot only sees channels its role can access.
Fix:
  1. Server Settings → Roles → create role "Bot"
  2. Enable: View Channels, Read Message History, Send Messages, Send in Threads, Embed Links, Manage Channels, Manage Messages
  3. Assign this role to your bot in Server Members
  4. For private channels: add the Bot role in that channel's Permission Overrides

View Channels Read Message History Send Messages Send in Threads Create Threads Embed Links Attach Files Add Reactions Manage Channels Manage Messages
